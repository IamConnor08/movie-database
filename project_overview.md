#Contributions

# Project Description
A movie database website that users can search and find information on movies and tv shows 

# Goals
Can search movies and tv shows by title, genre, year, or ratings.
displays ratings, cast, and poster.
allows users to add movies and tv shows to watchlist
frontend, backend, and database all work together properly

# Tools and TEchnologies
Frontend: React, HTML, CSS, JavaScript 
Backend: Python, FastAPI
Database: SQLite
API: The Movie Database
